CTR-structure is a scientific software for surface structure analysis. 
When you publish your paper by using this software, please cite

 M.Anada, Y.Nakabayashi-Ohno, M.Okada, T.Kimura and Y.Wakabayashi, ``Bayesian inference of metal oxide ultrathin film structure based on crystal truncation rod measurements,'' J. Appl. Cryst. (2017). 50, 1611-1616.

Usage of this software is explained in manual.pdf.
