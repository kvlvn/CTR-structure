CTR-structure is a scientific software for surface structure analysis. 
When you publish your paper by using this software, please cite

 M.Anada, Y.Nakabayashi-Ohno, M.Okada, T.Kimura and Y.Wakabayashi, ``Bayesian inference of metal oxide ultrathin film structure based on crystal truncation rod measurements,'' 
 J. Appl. Cryst.  50, 1611-1616 (2017).
 and
 K. Nagai, M. Anada, Y. Nakanishi-Ohno, M. Okada and Y. Wakabayashi
 "Robust surface structure analysis with reliable uncertainty estimation using the exchange Monte Carlo method"
 J. Appl. Cryst. 53 387-392 (2020).

Usage of this software is explained in manual.pdf.
